Scales information

Teaches a number of grade one scales. Green text indicates a correct note, and red indicates an incorrect note.

Compatibility

This project works on Chrome version 28+, and now also works on Firefox Nightly. It also works on Chrome for Mobile, version 31+.

Your browser will ask for permission to use the microphone once you press start.

If it doesn't work, check:

  • (Chrome) Click the camera button at the top right of the browser (in the address bar), and make sure that the correct microphone is selected.
  • (Firefox) Refresh the page, and turn live input on again. Make sure you selected the correct input device before clicking 'share'.
