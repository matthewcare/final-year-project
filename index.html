Sound analyser

Analyses sound input via microphone or audio mixer, and tells you what note the sound makes. The analyser works best with whistling at the moment, but I am working on making it more accurate for use with musical instuments.

This project only works on Chrome version 28+

Chrome will ask for permission to use the microphone at the top of the screen once you turn live input on.

If it doesn't work click the camera button at the top right of the browser (in the address bar), and make sure that the correct microphone is selected.
