Sound analyser

Analyses sound input via microphone or audio mixer, and tells you what note the sound makes.

This project works on Chrome version 28+, and now also works on Firefox Nightly.

Managed to get it to work with Chrome for Mobile, version 31+. It might also work for earlier versions, but I have no way of testing that at the moment

Your browser will ask for permission to use the microphone at the top of the screen once you turn live input on.

The value displayed on the left, is the live representation of the input. You will be able to see the different frequencies displayed in real time.

The value displayed on the right, is the 'true' representation of the input. It calculates the values based on the whole duration of the input.

If it doesn't work, check:

  • (Chrome) Click the camera button at the top right of the browser (in the address bar), and make sure that the correct microphone is selected.
  • (Firefox) Refresh the page, and turn live input on again. Make sure you selected the correct input device before clicking 'share'.
